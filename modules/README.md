# Structure of the modules directory

The modules directory contains the following subdirectories:

```
modules
├── Demo
│ ├── configs
│ │ └── demo.php
│ ├── migrations
│ ├── resources
│ │ └── languages
│ │ └── views
│ ├── routes
│ │ └── routes.php
│ ├── src
│ │ └── Commands
│ │ └── Http
│ │ | └── Controllers
│ │ | └── Middlewares
│ │ └── Models
├── Demo2
│ ├── ...
└── ModuleServiceProvider.php
```
